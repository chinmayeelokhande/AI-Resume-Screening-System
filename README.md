# AI Resume Screening & HR Analytics Dashboard

An advanced NLP-based resume screening system built with:

- TF-IDF Matching
- SBERT Semantic Matching
- HR Analytics Dashboard
- Streamlit Deployment

## Features

- Upload multiple resumes (PDF)
- Select job role
- Semantic similarity scoring using SBERT
- Shortlisting threshold control
- HR analytics dashboard
- Interactive visualizations

## Tech Stack

- Python
- Streamlit
- Scikit-learn
- Sentence Transformers (SBERT)
- Pandas
- PDFPlumber

## Run Locally

pip install -r requirements.txt  
streamlit run app.py
